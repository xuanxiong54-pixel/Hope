# 希望之锚 - 交互设计文档

## 核心交互理念
基于希望理论、习得性乐观理论和情绪叙事理论，为用户提供结构化的心理重建支持。

## 主要交互模块

### 1. 初始评估与引导模块
- **欢迎界面**: 温暖的问候语和共情表达
- **状态评估**: 通过温和问题了解用户当前心理状态
- **个性化引导**: 根据评估结果推荐合适的模块

### 2. 希望路径导航系统
- **目标设定器**: 
  - 引导用户设定具体、可衡量的小目标
  - 提供目标模板和建议
  - 实时验证目标的可行性
- **路径生成器**:
  - 为每个目标生成3条可选行动路径
  - 每条路径包含具体步骤和时间建议
  - 用户可以选择最适合的路径
- **进度追踪器**:
  - 可视化进度条显示完成情况
  - 能量槽系统激励持续行动
  - 里程碑庆祝和成就解锁

### 3. 认知重塑实验室
- **想法记录器**:
  - 安全私密的想法输入界面
  - 结构化的问题引导
  - 自动保存和隐私保护
- **ABCDE模型引导**:
  - A (Adversity): 记录困境事件
  - B (Belief): 识别自动化想法
  - C (Consequence): 分析情绪后果
  - D (Disputation): 证据辩论环节
  - E (Energization): 积极重构结果
- **重构助手**:
  - 引导式提问帮助审视想法
  - 提供多种解释角度的建议
  - 生成个性化的积极重构方案

### 4. 心灵故事屋
- **阶段化书写模板**:
  - "此刻的安顿" - 适用于情绪波动期
  - "情绪的河流" - 适用于情感表达期  
  - "故事的新篇章" - 适用于意义重构期
- **安全书写环境**:
  - 私密性承诺和保障
  - 自动保存草稿功能
  - 可选匿名分享至社区希望之树
- **书写引导工具**:
  - 分步骤的写作提示
  - 情绪词汇库和表达建议
  - 结构化模板帮助组织思绪

## 交互流程设计

### 用户进入流程
1. 温暖欢迎 → 2. 状态评估 → 3. 模块推荐 → 4. 具体功能引导

### 模块内交互循环
1. 功能介绍 → 2. 用户输入 → 3. 系统反馈 → 4. 深入引导 → 5. 进展总结

### 退出和持续支持
1. 进展总结 → 2. 积极反馈 → 3. 后续建议 → 4. 专业帮助提醒

## 关键交互原则
- **温暖共情**: 所有语言充满理解和支持
- **渐进引导**: 从简单到复杂，循序渐进
- **即时反馈**: 对每个操作给予积极回应
- **隐私保护**: 严格保障用户信息安全
- **文化敏感**: 体现集体主义价值观和"我们在一起"的理念
